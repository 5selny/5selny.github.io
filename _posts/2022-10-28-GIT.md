> ## 그 외

```
$ git branch 브랜치 확인
$ git checkout -b <이름> 브랜치 생성 및 이름 설정
$ git reset (--hard) 스테이징 취소
$ git log 커밋이력확인
$ git restore 커밋 전 첫 clone 상태로 돌아감
$ git remote -v 연결확인
$ git remote add (origin) (address) 연결
```
